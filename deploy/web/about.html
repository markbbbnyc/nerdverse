Every system — state, inventory, world events, companion agency — lives in MariaDB. The database is the save file.

THE TERMINAL

AS/400-style push/pop navigation. Green phosphor aesthetic. Full-screen feel. Nerdfont icons. Visible agency. Zero dependencies.
